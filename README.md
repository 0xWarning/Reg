# Help

Install Node.js **[Link](https://nodejs.org/en/download/)**
Do These Steps in a Terminal Type Enviroment
- cd Reg
- npm i
- mkdir includes
- cd includes
- nano config.json or edit via a form of notepad
- paste config code in and configure it to your liking
- save config.json
- cd ..
- node index.js
